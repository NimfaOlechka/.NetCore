# .NET Core Web Application

## Overview

This project is a personal ASP.NET Core application built to explore backend development, application structure, and working with data-driven systems.

The focus has been on building a clean and maintainable architecture while implementing common backend features such as authentication, database interaction, and structured business logic.

---

## Features

* User authentication and authorization
* Database integration using Entity Framework
* Structured application using services and repositories
* Basic data handling and business logic
* MVC / web application structure

---

## Tech Stack

* C# / .NET Core
* ASP.NET Core MVC
* Entity Framework Core
* SQL Server

---

## Purpose

The purpose of this project is to gain hands-on experience with:

* Backend development in .NET
* Structuring applications for maintainability
* Working with data flows and persistence
* Understanding how different layers of an application interact

---

## Notes

This project is intended as a learning and experimentation project, with focus on practical implementation and understanding of backend systems rather than production-ready features.

---

## How to Run

1. Clone the repository
2. Configure database connection
3. Run migrations
4. Start the application
